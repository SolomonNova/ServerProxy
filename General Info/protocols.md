# Networking Protocol Definitions

### Transport Layer Protocols
**Definition:** Governs **how** the bytes are sent.
It handles the mechanics of delivery, including reliability, error checking, and data flow between two endpoints (e.g., TCP or UDP).



---

### Application Layer Protocols
**Definition:** Governs the **meaning** of the bytes.
It defines the format and syntax of the data so that the specific application (like a web browser or email client) knows how to interpret the information (e.g., HTTP, SMTP, or FTP).
